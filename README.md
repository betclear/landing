# BetClear

Premium landing page for BetClear — an iPhone app that blocks gambling websites through iOS configuration profiles.

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- Deployed for Vercel

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` — local development
- `npm run build` — production build
- `npm run start` — serve production build
- `npm run lint` — ESLint

## Scope (MVP)

This repository currently ships the marketing site and a placeholder pricing page. Stripe checkout, success flow, and profile download are intentionally deferred to a follow-up milestone.
